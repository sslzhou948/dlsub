Privacy Policy

Last updated: May 16, 2026  |  Applies to: DL Subtitles Chrome Extension

Summary: DL Subtitles collects zero user data. Your API key is stored only on your own device and never sent to our servers (because we have none).

1. What We Collect

We collect nothing. DL Subtitles does not collect, transmit, store, or share any personal information, usage data, or analytics of any kind.

2. API Key Storage

When you configure an API key in the extension's Options page, it is stored exclusively in chrome.storage.sync — a local browser storage that Chrome may sync across your own signed-in devices. The key is:

  • Never sent to any server operated by this extension
  • Never logged or recorded
  • Used only to authenticate requests you initiate, sent directly from your browser to the API endpoint you configured

3. Translation Requests

When a subtitle cue is detected, the extension's Service Worker sends the subtitle text directly to the API endpoint you configured (e.g., OpenAI, DeepSeek, or your own proxy). This request:

  • Goes directly from your browser to your chosen API provider
  • Does not pass through any DL Subtitles server
  • Is governed by the privacy policy of the API provider you choose

4. Permissions Justification

Permission Why It's Needed
storage Save your API configuration and display preferences locally
scripting Inject the subtitle observer script into deeplearning.ai pages
learn.deeplearning.ai/* The only site where the extension activates
https://*/* Required so the Service Worker can reach any API endpoint you configure (BYOK — Bring Your Own Key). Without this, the extension could only call a fixed, hard-coded URL.

5. No Third-Party Sharing

We do not sell, trade, or share any data with third parties. There is no data to share.

6. Children's Privacy

This extension does not target children and does not knowingly process data from anyone under 13.

7. Changes to This Policy

If this policy ever changes, the updated version will be posted at this URL and the "Last updated" date above will be revised.
